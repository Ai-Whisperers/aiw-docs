## Features

### Multi-turn Conversation
Nyx remembers the last 20 messages in a conversation for 24 hours.

```
You: I'm working on a Python project
Nyx: Great! What kind of project is it?
You: A web scraper
Nyx: Nice! What website are you scraping?
```

### Voice Messages
Send a voice note and Nyx will:
1. Transcribe it
2. Understand the intent
3. Reply with text AND a voice note

### Images
Send an image with a caption and Nyx will:
1. Describe the image
2. Answer questions about it

### Web Search
Ask about recent events or information:
```
You: What are the latest AI developments?
Nyx: [searches the web] Based on recent news...
```

### Knowledge Base
Nyx has 36 entries covering:
- Team information
- Product details
- Common questions
- Technical documentation

### Multi-agent Routing
Nyx automatically routes to the best agent:
- **General**: Casual conversation
- **Teacher**: Explanations, tutorials
- **Researcher**: Deep dives, analysis
- **Coder**: Code questions, debugging

## Language Support
- Spanish (native)
- English
- Guarani

Nyx detects your language and responds in the same.

## Tips

1. **Be specific** — "help me debug this" is better than "help"
2. **Use commands** — `/hermes` for code questions routes to the code agent
3. **Voice first** — Try voice messages for quick queries
4. **Group mentions** — Both @nyx and @erebus work in groups
5. **Context matters** — Nyx remembers the conversation
